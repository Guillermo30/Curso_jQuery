Datos académicos

Objetivos

  • Aprender a utilizar los aspectos básicos de jQuery Mobile: Multipáginas, botones, cuadros de mensaje, popups, temas, e interacción con aplicaciones del propio dispositivo (sms, teléfono, mail)
  • Aprender a utilizar los componentes de la interfaz de usuario en jQuery Mobile: Barras de navegación, elementos colapsables, agrupaciones de componentes y efectos
  • Aprender a utilizar listas y sus componentes en jQuery Mobile: Listas inset, separadores de listas, listas maestro detalle, listas anidadas, autofiltrado con cuadros de búsqueda, listas con múltiples elementos y listas agrupadas
  • Aprender a utilizar formularios y sus componentes en jQuery Mobile: Formularios, sliders, listboxes, interruptores, radio buttons y check boxes

Contenidos

  • Aspectos básicos del framework jQuery Mobile 5 h
  • Componentes de la interfaz de usuario en jQuery Mobile 5 h
  • Listas en jQuery Mobile 5 h
  • Formularios en jQuery Mobile 5 h
  • Desarrollo de una aplicación de tienda online para dispositivo táctil 10 h

Metodología

  • Curso online mediante videotutoriales que muestran de forma práctica y sencilla cada una de las técnicas y controles de jQuery Mobile necesarios para desarrollar una aplicación para un dispositivo táctil
  • Tras cada videotutorial se plantea un problema a resolver basado en lo aprendido en el videotutorial
  • Tutorías on line y foros usando la plataforma del curso
